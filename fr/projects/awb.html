  • A propos

Ce projet concernant AutoWikiBrowser avait pour but de reprendre tous le code source du logiciel open source, qui permet de modifier les wikis comme Wikipédia ou autre. L'objectif en reprenant tous le code source, était de le modifier et de le transformer en logiciel français.

Le projet a été lancé en octobre-novembre 2020, mais par la suite, pour des sourcis de droit d'auteur, le projet fut abandonner en janvier 2021.

Mais ce projet n'est pas complètement abandonner, en effet avec .NET, les développeurs d'AutoWikiBrowser peuvent installer un système d'intertionnalisation (i18n), qui serait directement intégré au code source principal du logiciel sur SourceForge.
